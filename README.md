## 프로젝트 제목
뱀파이어 서바이벌(Vampire Survivors)


### 게임 컨셉
1) 한정된 구역
2) 저장 없음
3) 레벨 업 시 랜덤 아이템 획득

-> 로그라이크 장르


### 개발 범위
화면: 메인 화면, 게임 화면, 설정 화면

조작: 스마트폰 좌 하단(공격 버튼), 우 하단(이동 버튼), 그 외 터치식

이미지: 아군 캐릭터, 적 캐릭터, 보스 캐릭터, 아이템

난이도: 체력 시스템, 여러 아이템의 시너지

사운드: 배경음, 효과음, 타격음

UI: 체력 바, 플레이 시간, 경험치 바
